# 角色理解200问：附录题库

这份不是正文必填表。

它是`卡住时抽题用`的附录题库。

如果用户一上来就被要求回答200问，大概率会从创作变成考试。

更好的用法是：

```text
24问：角色还没轮廓时用。
80问：正式整理角色时用。
200问：某一块卡住时抽题用。
```

比如用户不知道怎么写调色盘，就抽`调色盘与衍生`部分。

不知道怎么写核心人格层，就抽`欲望、缺失、恐惧`和`防御与决策`部分。

不知道怎么防止AI写偏，就抽`二次解释与防误读`部分。

---

# 回答标准

每道题都尽量不要只答标签。

错误回答：

```text
她很缺爱。
```

更好的回答：

```text
她一旦发现自己开始期待{{user}}回复，就会故意晚一点回。
像是在证明自己没有那么需要对方。
```

回答时可以用这个格式：

```text
我的答案：
具体行为：
容易被AI误读成：
还没想明白：
```

# 第一组：角色核心

## 001. 这个角色一句话是什么？

解释：抓住角色最核心的可识别性。

参考回答：她是一个白天扮演乖乖大小姐、夜晚靠吉他找回自我的叛逆少女。

参考回答：必须保留“扮演与真实自我的冲突”，吉他可以换，冲突不能换。

解释：提前给AI设误读禁区。

参考回答：她不该被写成满嘴脏话的不良少女，她的脏话是情绪泄漏，不是日常标签。

参考回答：她公开得体，私下却会把所有电量砸进舞台。

解释：第一印象决定AI最容易抓住的外壳。

参考回答：看起来礼貌、漂亮、稳定，像不会出错的人。

解释：这是角色的第二层。

参考回答：熟悉后会发现她很懒散，也很容易在没人的地方松掉。

解释：这是隐藏层，不能对所有人开放。

## 009. 她是主动推动剧情的人，还是被剧情推动的人？

解释：决定角色有没有叙事主动权。

参考回答：她平常被环境推着走，但一碰到舞台和亲密关系会主动做决定。
