# 玩家手册

本目录保存玩家可见信息、角色创建说明和默认可玩化身入口。

## 目录用途

`manual/player-guide/` 是玩家的核心入口文档，包含角色创建流程、世界常识介绍、可选化身列表等。这里的内容应该对玩家友好、避免剧透，帮助新玩家快速进入游戏。

## 基本结构

可包含多个子目录，如 `playable-characters/`（可选化身）、`character-creation/`（角色创建指南）等。
