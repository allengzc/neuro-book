# 可选化身

本目录保存可直接开局使用的预设化身和玩家角色入口。

## 目录用途

`manual/player-guide/playable-characters/` 列出所有可供玩家选择的预设化身，每个化身是一个快速开局的入口，玩家可以直接使用或在此基础上修改。与 `simulation/subjects/` 不同，这里是"可选的角色模板"，不是"正在运行的角色状态"。

## 基本结构

每个化身是一个子目录或单文件，包含角色背景、初始状态、可见于玩家的设定等。
