# 说明书

本目录保存玩家可读手册、世界导览、规则指南、主持人手册和速查资料。

## 目录用途

`manual/` 存储面向玩家的公开说明文档，包括世界背景介绍、规则解释、角色创建指南等。与 `lorebook/`（全知视角设定）不同，这里的内容是玩家角色"应该知道"的常识，不包含剧透和秘密。

## 基本结构

子目录按手册类型组织，如 `player-guide/`（玩家指南）、`world-primer/`（世界入门）、`rules/`（规则说明）等。
