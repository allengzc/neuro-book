# 参考资料

本目录保存外部素材、导入归档、低置信迁移材料和原始来源摘录。

## 目录用途

`reference/` 存储从外部来源导入的素材、低置信设定、待整理的迁移材料等。这些内容的置信度低于 `lorebook/`，不应直接用作创作依据，需要先整理、验证后迁移到 `lorebook/` 或 `simulation/`。

## 基本结构

可按来源或类型组织子目录，如 `imported/`（导入素材）、`web-clippings/`（网页摘录）、`legacy/`（旧版迁移）等。
