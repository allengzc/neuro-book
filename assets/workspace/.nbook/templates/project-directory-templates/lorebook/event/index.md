# 事件

本目录保存已发生的历史事件、背景事件和重要时间节点。

## 目录用途

`lorebook/event/` 存储世界时间线上的重大事件，如战争、灾难、政变、发明等。这些是"已经发生"的背景事实，用于角色对话和剧情推演时的参考。不记录"正在发生"的剧情事件，当前剧情推演放入 `simulation/runs/`。

## 基本结构

子目录按事件类型或时间段组织，如 `wars/`、`disasters/`、`political-events/` 等。

## 命名约定

事件目录使用 kebab-case，如 `great-plague`、`regicide-of-752`。
