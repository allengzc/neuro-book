# 世界设定

本目录保存世界结构、历史、地理、文化、生态和世界内规则。

## 目录用途

`lorebook/world/` 存储宏观世界观设定，包括宇宙法则、历史进程、地理布局、文化体系、生态系统等全局性内容。与 `location/` 的具体地点不同，这里关注的是世界的整体框架和运行规律。

## 基本结构

可按主题组织子目录，如 `history/`、`geography/`、`culture/`、`magic-system/` 等，或根据项目需要自由划分。
