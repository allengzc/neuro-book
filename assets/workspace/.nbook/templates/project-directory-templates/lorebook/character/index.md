# 角色档案

本目录保存上帝视角角色设定、背景、秘密、关系和作者备注。

## 目录用途

`lorebook/character/` 存储全知视角的角色设定，包括角色的真实动机、隐藏背景、作者意图和剧情真相。与 `simulation/subjects/` 不同，lorebook 角色是无状态原型，不追踪剧情中的记忆和心理变化。

## 基本结构

子目录按角色名组织，每个角色可包含多个设定文件（如背景、关系、秘密等）。

## 命名约定

角色目录使用 kebab-case，如 `baron-brauer`、`elder-maid`。
